Data is sourced using the covidregionaldata package to download data published by Public Health Infobase, Public Health Agency of Canada.

The data has been processed using the EpiNow2 R package to estimate the time-varying reproduction number, rate of spread, and doubling time. EpiNow2 uses a range of open-source tools (Abbott et al.), and current best practices (Gostic et al.).

The visualisation is built using the RtD3 package following the example given for subnational results.
